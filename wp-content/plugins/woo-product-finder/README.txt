Product Finder for WooCommerce let customers narrow down the product list on the basis of their choices.and user can find easily any kind of product on store.

== Description ==

<a href="https://bit.ly/39BnC9P" target="_blank">Product Finder for WooCommerce</a> let customers narrow down the product list on the basis of their choices. 

It enables the store owners to add a questionnaire to the product page.

The Product Finder are then rendered according to the answers, given by the users and you can showcase 'n' number of products, matching the answers and query. 

Pro Plugin Demo : <a href="http://pluginsdemo.thedotstore.com/woocommerce-product-finder/" target="_blank">View Demo</a>

= HOW IT WORKS? =
1. Before add wizard you will have to select product attribute in particular product.
2. Enter Wizard title, and enable status. 
3. Manage Question: Add New Question( Create question option like a radio button ) 
4. Copy and past Wizard shortcode in page/ post for show wizard.

= Key Features =

* Create Wizard (Questions and answers) 
* Manage Question: Add New Question 
* Manage Options: Add New Attribute Name and value.
